## Phase Goal

**As a** D&D player whose character just leveled up, **I want to** see a clear, immediate visual card in the chat above the narration showing my new level, my HP increase (old → new), and (at level 5) my new proficiency bonus, **so that** I understand exactly what mechanical reward I just earned without having to re-read the narrative or check the Party tab.

<objective>
Render a new `LevelUpCard` React component in the existing `FieldTab` chat slot, fed by `levelUpResult` returned from `take-turn.ts` (Plan 03). The card lives alongside the existing `DiceCard` in the same `space-y-2` container — same visual pattern, different color family (indigo for level-up vs amber for dice). State is cleared on turn start (parallel to `setDiceResult(null)`) and set on turn success.

Purpose: Completes the visible end of the LVL-02 vertical slice — the player FINALLY sees a UI card confirming their HP and level changed. Phase 2 deliberately deferred this notification (D-07 in Phase 2 CONTEXT.md); Phase 3 D-05/D-06/D-07 closes that gap with the same DiceCard pattern proven in Phase 1.

Output: One edited file (app/game/[id]/page.tsx) with five surgical additions: LevelUpResult type declaration, `levelUpResult` state, three handler updates (clear-on-start, set-on-success, clear-on-failure), `FieldTab` prop, and the LevelUpCard component + its render slot. Plus a human-verify checkpoint to confirm the visual.
</objective>
